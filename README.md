# 项目介绍
- 使用高德api
- 鼠标点击坐标点显示该点坐标
- 坐标点连线

# 运行
1、ide导入该项目或者在控制台打开文件夹map

2、控制台输入 `npm install`,等待依赖添加完成

3、依赖安装完成后在控制台输入`npm run serve`，等待，点击控制台显示的本地运行地址

> **由于开发时使用的是vue3的版本，运行时可能存在一些问题，如果出现页面加载不出但没有报错的情况，请把MapContainer.vue的内容用下面的代码替换**
